# Spam Message / Email Detection using Machine Learning

This project is a **Machine Learning application** that detects whether a message or email is **Spam** or **Ham (Not Spam)** using supervised learning.

## Features
- Preprocess text data using NLP
- Convert text to numerical features with TF-IDF
- Train a Naive Bayes classifier
- Evaluate model with accuracy, precision, recall, and confusion matrix
- Test predictions on new messages

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
